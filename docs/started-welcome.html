兼容性

Owl Carousel 2.x.x 不兼容旧版本 1.x.x. 它与 Owl1 有很多共同点，保持想法一致。 它是根据Owl1的核心代码基础上重新编写的，我为它的新功能感到荣幸。

Owl Carousel 测试支持如下浏览器或设备：

  • Chrome
  • Firefox
  • Opera
  • IE7/8/10/11
  • iPad Safari
  • iPod4 Safari
  • Nexus 7 Chrome
  • Galaxy S4
  • Nokia 8s Windows8

Library

下载适合您需求的版本：

  • Owl Carousel - 2.0.0-beta.2.4 - 分布式版本 - 已编译压缩 包含Javascript, images and CSS 。
  • Owl Carousel Source - 2.0.0-beta.2.4 - 源文件，包括本文档。 全部包裹在 Grunt 项目。
