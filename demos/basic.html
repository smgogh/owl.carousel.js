概要

猫头鹰旋转木马的基本用法。我用 loop:true and margin:10. 该结构适用于任何类型的DOM元素。在我的所有例子中，我用 <div class="item">...</div> 但你可以用到任何其他元素 div/span/a/img...

设置

$('.owl-carousel').owlCarousel({
    loop:true,
    margin:10,
    nav:true,
    responsive:{
        0:{
            items:1
        },
        600:{
            items:3
        },
        1000:{
            items:5
        }
    }
})
